# Подключение к базе данных SQLite

Была реализована базовая работа с `SQLite`. С помощью метода `openOrCreateDatabase()` создана база данных `app.db` и таблица `users`. Далее в таблицу добавлены записи с использованием SQL-запроса `INSERT`, после чего выполнен запрос `SELECT` для получения данных. Результаты запроса обработаны с помощью `Cursor` и выведены в `TextView`. В конце работы ресурсы (база данных и курсор) были корректно закрыты.
