# SQLiteOpenHelper и SimpleCursorAdapter, получение данных из SQLite

Для организации доступа к базе данных реализован вспомогательный класс `DatabaseHelper`, наследуемый от `SQLiteOpenHelper`. В данном классе переопределены методы `onCreate()` и `onUpgrade()`: при создании базы данных формируется таблица `users`, содержащая поля `_id`, `name` и `year`, а также добавляется начальная запись; при обновлении версии базы данных выполняется её пересоздание.

Пользовательский интерфейс приложения включает элементы `TextView` и `ListView`, предназначенные для отображения количества записей и списка данных соответственно.

В классе **MainActivity** реализовано подключение к базе данных с использованием метода `getReadableDatabase()`. Извлечение данных осуществляется посредством SQL-запроса через метод `rawQuery()`, результат которого представлен объектом Cursor. Для отображения данных в списке применён `SimpleCursorAdapter`, обеспечивающий связывание данных из курсора с элементами интерфейса.

Завершение работы с ресурсами организовано в методе `onDestroy()`, где выполняется закрытие базы данных и курсора.

В результате приложение обеспечивает корректное получение и отображение данных из базы **SQLite** в виде списка.
