# Использование существующей БД SQLite
Создание и подготовка базы данных. 

База данных была заранее создана во внешнем инструменте (**SQLite**). В ней определена таблица users с полями: `_id`, `name`, `year`. В таблицу добавлены начальные данные.
На основе реализованного в предыдущем задании функционала (добавление, редактирование и удаление записей) была реализована работа с заранее созданной БД.
