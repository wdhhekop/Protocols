# Динамический поиск по базе данных SQLite

Был изучен механизм фильтрации с использованием `SimpleCursorAdapter`, а также обработка пользовательского ввода через `TextWatcher`.

В процессе работы рассмотрено, как при изменении текста в поле ввода выполняется фильтрация данных с помощью метода `filter()`, а также как с помощью `FilterQueryProvider` реализуется SQL-запрос с условием `LIKE` для поиска записей по введённой строке. Кроме того, изучена работа с существующей базой данных, подключённой из папки assets, и отображение отфильтрованных данных в списке `ListView`.
